# TypeScript

## AI-era hypothesis

TypeScript is well-positioned because it combines JavaScript ecosystem gravity with stronger machine-readable structure.

## Strengths

- Massive ecosystem
- Strong editor and language-server support
- Gradual static typing
- Good fit for web and full-stack applications
- High AI-training representation

## Weaknesses

- Type-system complexity
- JavaScript runtime semantics remain underneath
- Dependency ecosystem risk
- Build tooling complexity

## Open questions

- Does TypeScript become the default agent-friendly application language?
- How much do JavaScript semantics constrain long-term reliability?
