# Framework Versions

## v0.1

Initial framework with five dimensions:

- Human cognition
- Machine cognition
- AI-agent operability
- Runtime and ecosystem
- Strategic viability

## v0.2

Added `ai_systems_interoperability` as the 6th dimension. Rebalanced weights so the new dimension carried 15%.

## v0.3

Added `structured_output_maturity` as the 7th dimension. Rebalanced weights to carry 10% for the new dimension by pulling from machine cognition and AI-systems interoperability.

## v0.4

Restructured the AI-era dimensions to separate native (language-steward-shipped) from ecosystem (third-party / community / commercial) signals, and added a dependency-risk dimension. Dimension count: 10. New weighting:

- HC 15%
- MC 15%
- AO 20%
- RE 10% (was 15)
- SV 10%
- AIN 7.5% (split from AI-systems interoperability 15%)
- AIE 7.5% (split from AI-systems interoperability 15%)
- SON 5% (split from structured-output maturity 10%)
- SOE 5% (split from structured-output maturity 10%)
- EDR 5% (new — funded from RE)
