# Source Notes

This file is reserved for freeform research notes that have not yet been promoted to atomic claims. The current corpus is fully promoted: every observation is a claim with a primary-source citation in the per-language registries (`<lang>-sources.yaml`).

If a future research pass collects observations that are not yet ready to be claims (open questions, conflicting reports, claims pending verification), they should be drafted here first and then refactored into atomic claims following the workflow in `outputs/evidence-backed-research-execution-plan.md`.
